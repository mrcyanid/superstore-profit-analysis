\# 🛒 Superstore Profit \& Discount Analyzer



\## 📊 Project Overview

This project analyzes how different discount levels impact profit margins across product categories in a retail superstore. The goal is to identify the exact discount threshold where profits turn negative for each category.



\## 🎯 Business Problem

Management was offering heavy discounts across all product categories without understanding the impact on profitability. This project provides data-driven recommendations to optimize discount strategies.



\## 🔧 Technologies Used

\- \*\*Python 3.8+\*\* - Core programming language

\- \*\*Pandas\*\* - Data manipulation and analysis

\- \*\*NumPy\*\* - Numerical computations

\- \*\*Matplotlib \& Seaborn\*\* - Data visualization

\- \*\*SQLite\*\* - Database storage (bonus feature)
